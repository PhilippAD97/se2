## Configuring Git in Eclipse
1) Click `Preferences` > `Team` > `Git` > `Configuration`
2) Click `New Entry` and enter the key value pairs:
    - Key: **user.name**, Value: **YourUsernameHere**
    - Key: **user.email**, Value: **YourEmailHere**
